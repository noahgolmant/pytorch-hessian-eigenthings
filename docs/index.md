# hessian-eigenthings

Iterative eigendecomposition of curvature operators (Hessian, GGN, Fisher) for PyTorch models.

This site documents the v1 alpha. See the [GitHub repository](https://github.com/noahgolmant/pytorch-hessian-eigenthings) for source.
